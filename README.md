# Project Dev Tools
    Java 11
    Node 16
    Yarn 1.22

# How to run
## Client
    cd src/main/javascript
    yarn install
    yarn start

## Server
    Change the location of spring.datasource.url to a local path in application.yml
    Run TradeSimulatorApplication
